# 📖 Data Dictionary — Coffee Shop Sales

Column definitions for `data/sales_data.csv`, generated by `generate_data.py`.

---

## Source Table Columns

| Column | Data Type | Description | Example |
|--------|-----------|-------------|---------|
| `Date` | String → Date | Transaction date (`YYYY-MM-DD`) | `2023-01-11` |
| `Item` | String | Product name — one of 10 items | `Latte` |
| `Quantity` | Integer | Units purchased in the transaction (1–5) | `2` |
| `Price` | Float | Per-unit price in USD | `4.75` |
| `Total` | Float | `Quantity × Price` — line revenue | `9.50` |

---

## Derived Columns (added in `analysis.py`)

| Column | Source | Logic | Purpose |
|--------|--------|-------|---------|
| `Month` | `Date` | `pd.to_datetime(Date).dt.to_period('M')` | Groups by calendar month |

## Data Quality Rules Applied

| Check | Action |
|-------|--------|
| `Total <= 0` | Row removed (voided transaction) |
| `Item` or `Total` is null | Row dropped |

---

## Generation Parameters

| Parameter | Value |
|-----------|-------|
| Date range | 2023-01-01 → 2023-03-31 (91 days) |
| Transactions per day | 20–100 (random integer) |
| Quantity per line | 1–5 units |
| Random seed | `42` — fully reproducible |
